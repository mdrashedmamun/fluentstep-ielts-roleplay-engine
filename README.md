# FluentStep: IELTS Roleplay Engine

Interactive IELTS roleplay scenarios with natural AI-powered pronunciation audio. Practice speaking with realistic conversations in social, workplace, service, and advanced contexts.

**Live**: https://fluentstep-ielts-roleplay-engine.vercel.app

## Quick Start

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```
3. Open http://localhost:5173
