# Screenshot Archive

Historical screenshots used as test evidence and deployment verification.

## Directory Structure
- `archive/` - All historical screenshots organized by feature/test session

## Guidelines
- Screenshots are automatically archived after 30 days
- New screenshots: `{feature}_{date}_{description}.png`
- Active screenshots (referenced in docs) stay in archive/
- Unused screenshots >90 days old can be deleted
