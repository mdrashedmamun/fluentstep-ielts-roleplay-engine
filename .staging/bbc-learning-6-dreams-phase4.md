## Transfer Learning Guidance

**Use this pattern summary to transfer learning to NEW scenarios:**

1. **Conversation Openers** → Use in hotel check-in, restaurant reservation, work meeting
2. **Validation Chunks** → Use when friends share good news OR bad news
3. **Idioms** → Collect idioms for each topic area (travel idioms ≠ work idioms)
4. **Topic-Specific Vocabulary** → Create bucket B vocabulary for each scenario
5. **Obstacle Naming** → Essential when discussing ANY change or difficulty
6. **Conditional Questions** → Use when inviting any vulnerable sharing
7. **Grammar/Function** → These are universal—master once, reuse everywhere
